# 📚 Sistema de Cadastro de Alunos em C

Este é um projeto modular desenvolvido em linguagem C para gerenciar o cadastro e a listagem de alunos, utilizando persistência de dados em arquivos binários.

## 👤 Sobre Mim
Olá! Eu sou o Tassio. Atualmente estou no **2º semestre de Engenharia de Software**.  
Este projeto foi desenvolvido como uma forma de treinar meus conhecimentos em:
- Lógica de programação com C.
- Manipulação de Arquivos (`FILE`, `fwrite`, `fread`).
- Organização de código modularizada (`.h` e `.c`).
- Versionamento com Git e GitHub.

## 🚀 Funcionalidades
- **Cadastro de Alunos:** Salva ID, Nome e Média em um arquivo `.dat`.
- **Listagem de Alunos:** Lê e exibe todos os alunos cadastrados de forma organizada.
- **Persistência de Dados:** As informações não se perdem ao fechar o programa.
